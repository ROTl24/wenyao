## 4. 安全与隐私验收

- Electron 启用 `contextIsolation`、禁用 `nodeIntegration`、启用沙箱和 Web 安全。
- 渲染进程仅通过窄 preload API 调用本地服务。
- API 密钥使用 Windows DPAPI 加密，公开设置接口不返回密钥密文或明文。
- 非本机 API 地址强制使用 HTTPS。
- 发行包内含预构建向量，日常分析只发送当前问题、不可变排盘和少量候选证据；手动重建向量时会明确提示古籍片段将分批发送给阿里向量模型。
- AI 返回的卦名、干支、动爻和证据 ID 会在本地校验，拒绝伪造引用或篡改排盘事实。
- 历史记录保存在当前 Windows 用户目录，写入采用临时文件加原子替换。
